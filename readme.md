# ЦОВО — Боковая панель

Прототип боковой панели для центра обработки входящих обращений (ЦОВО).
Верстка передаётся разработчику как визуальный шаблон — функциональные заглушки на JS демонстрируют поведение интерфейса при взаимодействии.

---

## Экраны

| Экран           | Описание                                                                                                         | Ссылка                                       |
| --------------- | ---------------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| Входящий звонок | Карточка входящего обращения, список пациентов, переключение вкладок «Обращения / Визиты»                        | [Открыть](https://vitny.github.io/COVO/call) |

```

├── web.html

```

## Реализованное поведение (JS-заглушки)

- **Навигация** — перемещение фона активного режима работы между кнопками в двух рядах
- **Табы** — переключение активного таба с подсветкой иконки
- **Список пациентов** — выбор/снятие выбора пациента, появление вкладки «Визиты»
- **Вкладки пациента** — переключение между «Обращениями» и «Визитами» с прокручиваемым контентом
- **Фильтр источников** — выпадающий список с чекбоксами, закрытие по клику вне области
- **Поиск** — выпадающий список результатов при фокусе, прокрутка при более 5 результатов, закреплённое предупреждение снизу
